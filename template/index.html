Fresno State Class Info

Welcome to Fresno State Class Info (FSCI), our goal is to help you better understand Fresno State's class offerings. This website is intended for Future as well as Current students. The information provided here can assist students in at least these ways:

  • Better inform students on which electives to choose.
  • Which classes are complementary and/or relavent in the work force.
  • Inform which courses staisfy graduation requirements.
  • Provide additional background information, and resources on the course and instructor.
